A set of utilities classes built with [Trumman](https://github.com/trummancss/trumman) and [CSS comments](https://github.com/filipelinhares/css-comments.git)

## classes
#### Alignment
- `.u-alignBoth`
- `.u-verticalAlign`
- `.u-noMargin`
- `.u-noPadding`
- `.u-marginBot[5-100]`
- `.u-marginTop[5-100]`
- `.u-marginLft[5-100]`
- `.u-marginRgt[5-100]`
- `.u-paddingBot[5-100]`
- `.u-paddingTop[5-100]`
- `.u-paddingLft[5-100]`
- `.u-paddingRgt[5-100]`

#### Layout
- `.u-clearfix`
- `.u-floatLeft`
- `.u-floatRight`

#### Text alignment
- `.u-textRgt`
- `.u-textLft`
- `.u-textCenter`
- `.u-textTruncate`

#### Text transform
- `.u-textUp`
- `.u-textLow`
- `.u-textCap`
- `.u-textNone`

#### Misc
- `.u-imageReplace`

#### Print
- `.u-noPrint`
- `.u-printable`
